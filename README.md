# Science, Faith & Dialogue

**Science, Creation, Contingency, and the Human Person**
*An Interactive Scholarly Resource for Buddhism-Christianity Dialogue*

## About

This website is an interactive scholarly resource exploring intersections between Christian philosophy of science, Buddhist philosophy, and natural science. It accompanies the lecture "Science, Creation, Contingency, and the Human Person" delivered at the Tran Nhan Tong Institute, Vietnam National University (Hanoi) in March 2026.

## Project Context

Part of the **Erasmus+ KA171** international mobility partnership between:
- **Evangelical Lutheran Theological Faculty**, Comenius University in Bratislava, Slovakia
- **Tran Nhan Tong Institute**, Vietnam National University, Hanoi

Joint research project: *"Exploring Intersections: Buddhism, Christianity, Confucianism, and Natural Science in the Age of AI"*

## Tech Stack

- **Astro 5** — Static site generation
- **React 19** — Interactive components (islands architecture)
- **Tailwind CSS 4** — Utility-first styling
- **TypeScript** — Type-safe data and components
- **GitHub Pages** — Deployment via GitHub Actions

## Development

```bash
npm install
npm run dev      # Start dev server
npm run build    # Build for production
npm run preview  # Preview production build
```
